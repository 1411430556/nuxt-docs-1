```bash [Terminal]
npx nuxi info [rootDir]
```

`info` 命令记录当前或指定 Nuxt 项目的相关信息。

选项        | 默认值          | 描述
-------------------------|-----------------|------------------
`rootDir` | `.` | 目标应用程序的目录。
