一个 `.gitignore` 文件指定了 git 应该忽略的故意未跟踪的文件。

:read-more{icon="i-simple-icons-git" color="gray" title="Git 文档" to="https://git-scm.com/docs/gitignore" target="_blank"}

我们建议至少在 `.gitignore` 文件中包含以下条目：

```bash [.gitignore]
# Nuxt 开发/构建输出
.output
.data
.nuxt
.nitro
.cache
dist

# Node 依赖项
node_modules

# 日志
logs
*.log

# 杂项
.DS_Store

# 本地环境文件
.env
.env.*
!.env.example
```
