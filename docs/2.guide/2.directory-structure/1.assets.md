该目录通常包含以下类型的文件：

- 样式表 (CSS, SASS 等)
- 字体
- 不会从 [`public/`](/docs/guide/directory-structure/public) 目录提供的图像。

如果您希望从服务器提供资产，建议查看 [`public/`](/docs/guide/directory-structure/public) 目录。
